# SQL Query Performance Analytics

## Project Overview

This project analyzes SQL query logs using Python, MySQL, and Power BI to monitor query performance, resource usage, and failed queries.

The project includes:

* Data Cleaning
* SQL Analysis
* Exploratory Data Analysis (EDA)
* Power BI Dashboard

---

# Tools Used

* Python
* Pandas
* NumPy
* MySQL
* Power BI

---

# Key KPIs

* Total Queries: 1601
* Success Rate: 58.78%
* Failed Queries: 336
* Avg Execution Time: 1.12 sec
* Avg CPU Usage: 15.79%
* Avg Memory Usage: 29.43 MB

# Key Insights

* UPDATE queries were executed most frequently.
* IT Support generated the highest query volume.
* Failed and TIMEOUT queries indicate performance issues.
* CPU and memory usage remained stable overall.
* Query activity changed across different months.

---

# Business Recommendations

* Optimize slow-running queries.
* Reduce failed query executions.
* Monitor high resource-consuming operations.
* Improve query validation and monitoring systems.
